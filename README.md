# Weather-App
A simple Weather App where you can check the weather of any city. You can see the temperature, humidity, description, and a weather emoji.
This app uses the OpenWeatherMap API to fetch real-time weather data.

# How to Use

1- Open index.html in your browser.

2- Type the city name in the input box.

3- Click Get Weather.

4- The card will show the city info, temperature, humidity, description, and emoji.

5- If the city is wrong or empty, an error message will appear.
